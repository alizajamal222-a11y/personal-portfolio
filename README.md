# Personal Portfolio Website

## Purpose
This portfolio showcases my skills, projects, and contact information. It is designed for potential employers and clients to view my work.

## Features
- Home page with an introduction.
- About Me section.
- Projects section with links.
- Contact page with a working form.
- Responsive design using HTML and CSS.

## Running Locally
1. Clone the repository or download the ZIP file.
2. Open the `index.html` folder in any modern browser.
